Cel aplikacji:
Pobranie aktualnych danych GPS użytkownika

Opis wykorzystanych danych z urządzenia:
expo-location

Opis wykorzystanych bibliotek i API:
expo-location do pobierania współrzędnych.
Open-Meteo API do pobierania pogody.

Opis przepływu danych w aplikacji:
1. Pobranie zgody na lokalizację.
2. Pobranie współrzędnych GPS.
3. Zapytanie do API o dane pogodowe.
4. Prezentacja danych w UI.

Lista ograniczeń i problemów napotkanych podczas realizacji:
Aplikacja wymaga aktywnego połączenia z Internetem.
